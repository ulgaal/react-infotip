import React from 'react' import Balloon from './Balloon'

The default value of the config property used by Source and Storage

export const defaultConfig = { position: { my: 'top-left', at: 'bottom-right', target: false, adjust: { mouse: false, x: 0, y: 0, method: 'none' }, container: null }, show: { delay: 0 }, hide: { delay: 0 }, wrapper: Balloon, wrapperProps: {} }

The library uses the ConfigContext to define configuration globally for a whole range of Source elements.

export const ConfigContext = React.createContext(defaultConfig)

This context is used internally for communication between Storage and its Source elements.

export const StorageContext = React.createContext(null)
